# Teste Rápido do PWA

## 1. Build e Start
```bash
cd frontend
npm run build
npm run start
```

## 2. Abrir no Navegador
```
http://localhost:3004
```

## 3. Verificar Service Worker
1. Abra DevTools (F12)
2. Application tab
3. Service Workers → deve aparecer "activated and is running"
4. Manifest → deve mostrar informações do app

## 4. Testar Offline
1. Application > Service Workers
2. Marque checkbox "Offline"
3. Refresh a página
4. Deve funcionar!

## 5. Instalar
Desktop: Clique no ícone "+" na barra de endereço

Mobile:
- Android: Menu > "Instalar app"
- iOS: Compartilhar > "Adicionar à Tela de Início"

## ✅ Pronto para Testar!

Se tudo funcionar, o sistema está pronto para ser um PWA!
